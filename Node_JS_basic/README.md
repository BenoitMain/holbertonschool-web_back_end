🚀 Getting Started

📥 Clone this repository

git clone <your-repo-url>
cd <your-repo-name>


📦 Install dependencies

npm install


▶️ Start the server

node server.js


Or with Nodemon (auto-reload):

nodemon server.js

🧪 Running Tests
npx mocha
